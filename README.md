# Tampermonkey Universal AI Prompt Commands IT

Script Tampermonkey per sostituire comandi brevi con prompt IA già pronti nelle chat di intelligenza artificiale.

Lo script sostituisce solo i comandi esatti con prompt preparati. Il testo normale non viene modificato. ChatGPT, Gemini, Claude, Copilot.

## Comandi

- `IT1` — Traduci integralmente e fedelmente il testo fornito in italiano.
- `IT2` — Riassumi il testo fornito in italiano in base al significato e al contesto.
- `IT3` — Crea in italiano un breve riassunto tematico della lettera rigorosamente in una sola riga.
- `IT4` — Traduci il testo fornito in tedesco semplice e chiaro, livello A2-B1.
- `IT5` — Correggi il testo fornito in italiano.
- `IT6` — Scrivi una risposta breve, cortese e formale a questa lettera in tedesco semplice, livello A2-B1.
- `IT7` — Spiega con parole semplici in italiano che cosa significa il testo.
- `IT8` — Estrai tutti i fatti importanti dal testo e organizzali in italiano: persone, organizzazioni, indirizzi, date, scadenze, importi, numeri di documenti, richieste, decisioni, obblighi, documenti e passi successivi.
- `IT9` — Crea in italiano un elenco chiaro delle azioni necessarie in base al testo.
- `IT10` — Redigi dal testo una lettera ufficiale, cortese e chiara in tedesco, livello A2-B1.

## Installazione

Installa Tampermonkey, apri il collegamento dello userscript, conferma l’installazione e inserisci il primo comando in una chat IA.

[Userscript](https://raw.githubusercontent.com/1777maxim7771/it_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js)

Lo script viene installato nell’estensione Tampermonkey, non in GitHub né in un sito. GitHub conserva soltanto il file.

Tampermonkey → Dashboard → Utilities → Import from URL:

`https://raw.githubusercontent.com/1777maxim7771/it_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js`
